0.2. Первая программа

Начнем с классики — вывода текста на экран. В Python это делается через команду print():


print("Hello, world!")  # Выведет: Hello, world!
Важно запомнить:
  • Текст в кавычках " " или ' ' называется строкой
  • Скобки () обязательны в Python 3!

0.3. Ввод данных

Чтобы получить данные от пользователя, используйте input():


name = input("Как тебя зовут? ")  # Пример: Как тебя зовут? Маша
print("Привет,", name)            # Выведет: Привет, Маша
Важно: input() всегда возвращает строку. Чтобы работать с числами, преобразуйте тип: 

age = int(input("Сколько тебе лет? "))  # Пример: Сколько тебе лет? 25
print("Через год тебе будет", age + 1)  # Выведет: Через год тебе будет 26

1. Операторы

1.1. Арифметические операторы

Python поддерживает все базовые математические операции:

Оператор Описание Пример Результат
+ Сложение 5 + 3 8
- Вычитание 10 - 4 6
* Умножение 3 * 4 12
/ Деление 10 / 2 5.0
// Целочисленное деление 10 // 3 3
% Остаток от деления 10 % 3 1
** Возведение в степень 2 ** 3 8 

print(5 + 3)    # 8
print(10 % 3)   # 1
print(2 ** 3)   # 8
Важно: Результат деления / всегда float, даже если числа делятся нацело

1.2. Приоритет операций

Операции выполняются в следующем порядке:

  1. Скобки ()
  2. Возведение в степень **
  3. Умножение, деление, остаток * / % //
  4. Сложение и вычитание + -

print(2 + 3 * 4)      # 14 (3*4=12 → 2+12=14)
print((2 + 3) * 4)    # 20 (скобки меняют приоритет)
Осторожно! Операции с одинаковым приоритетом выполняются слева направо

1.3. Операторы сравнения

Оператор Описание Пример Результат
== Равенство 5 == 5 True
!= Неравенство 5 != 3 True
> Больше 10 > 5 True
< Меньше 3 < 5 True
>= Больше или равно 5 >= 5 True
<= Меньше или равно 4 <= 5 True 

print(5 == 5)   # True
print(10 < 5)   # False

1.4. Логические операторы

Оператор Описание Пример Результат
and Логическое И True and False False
or Логическое ИЛИ True or False True
not Логическое НЕ not True False 

age = 25
print(age > 18 and age < 30)  # True (25 между 18 и 30)
print(not age > 30)           # True (25 не больше 30)

2. Условный оператор

2.1. Базовый синтаксис

Условный оператор if позволяет выполнять код при соблюдении условия:


if True:
    print("Этот код выполнится всегда")
Важно: Отступы (4 пробела) обязательны в Python!

2.2. Блок else

Выполняется, когда условие ложно:


age = 15
if age >= 18:
    print("Доступ разрешен")
else:

score = 85
if score >= 90:
    print("Отлично")
elif score >= 75:

    print("Удовлетворительно")
else:
    print("Не сдано")
Внимание: Порядок условий в elif важен!

3. Цикл for

3.1. Основы цикла for

Цикл for используется для перебора последовательностей (списков, строк, словарей и т.д.):


fruits = ["яблоко", "банан", "вишня"]
for fruit in fruits:
    print(fruit)
Отличие от while: for используется когда известно количество итераций

3.2. Функция range()

Генерация числовых последовательностей:


# От 0 до 4 (5 не включается)
for i in range(5):
    print(i)  # 0,1,2,3,4

# От 2 до 8 с шагом 2
for i in range(2, 10, 2):
    print(i)  # 2,4,6,8

3.3. Перебор с индексами

Использование enumerate():


fruits = ["яблоко", "банан", "вишня"]
for index, value in enumerate(fruits):
    print(f"Индекс {index}: {value}")

3.4. Вложенные циклы


for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")
Осторожно! Следите за отступами во вложенных циклах

3.5. Работа со строками


text = "Python"
for char in text:
    print(char)
    
# Вывод:
# P
# y
# t
# h
# o
# n

3.6. Прерывание цикла


for i in range(10):
    if i == 5:
        break
    print(i)  # Выведет 0-4

3.7. Пропуск итерации


for i in range(5):
    if i == 2:
        continue
    print(i)  # Выведет 0,1,3,4

3.8. Генераторы списков

Создание списков в одну строку:


squares = [x**2 for x in range(5)]
print(squares)  # [0, 1, 4, 9, 16]

4. Цикл while

4.1. Базовый синтаксис

Цикл while выполняется пока условие истинно:


count = 0
while count < 5:
    print(count)
    count += 1  # Без этой строки цикл будет бесконечным
Опасность: Бесконечный цикл может повесить программу!

4.2. Управление циклом

Операторы break и continue:


# Прерывание цикла
i = 0
while True:
    print(i)
    i += 1
    if i == 3:
        break  # Выйдет при i=3

# Пропуск итерации
j = 0
while j < 5:
    j += 1
    if j % 2 == 0:
        continue
    print(j)  # Выведет 1,3,5

4.3. Бесконечные циклы с пользовательским вводом


while True:
    command = input("Введите команду (exit - выход): ")
    if command == "exit":
        break
    print(f"Вы ввели: {command}")

4.4. Обработка ошибок в циклах


while True:
    try:
        num = int(input("Введите число: "))
        break
    except ValueError:
        print("Это не число! Попробуйте еще")

4.5. Вложенные циклы while


i = 1
while i <= 3:
    j = 1
    while j <= 3:
        print(f"i={i}, j={j}")
        j += 1
    i += 1

4.6. Имитация do-while

Выполнение хотя бы одной итерации:


while True:
    number = int(input("Введите положительное число: "))
    if number > 0:
        break

5.3. Методы списков

Добавление элементов


nums = [1, 2]
nums.append(3)       # [1, 2, 3]
nums.insert(1, 1.5)  # [1, 1.5, 2, 3]
nums.extend([4, 5])  # [1, 1.5, 2, 3, 4, 5]

Удаление элементов


nums = [1, 2, 3, 4, 5]
nums.pop()     # Удаляет последний элемент → 5
nums.pop(1)    # Удаляет элемент с индексом 1 → 2
nums.remove(3) # Удаляет первое вхождение 3
del nums[0]    # Удаляет по индексу

5.4. Сортировка и реверс


arr = [3, 1, 4, 1, 5, 9]
arr.sort()          # Изменяет список → [1, 1, 3, 4, 5, 9]
sorted_arr = sorted(arr)  # Создает копию
arr.reverse()       # Реверсирует список → [9, 5, 4, 3, 1, 1]

6. Функции

Функции — это основной инструмент для организации кода в Python. Они позволяют избежать дублирования, упрощают чтение программы и делают её более модульной. В этой главе мы разберёмся, как создавать функции, передавать аргументы, возвращать значения и решать задачи с их помощью.

6.1. Основы создания функций

Функция определяется с помощью ключевого слова def, за которым следует имя функции и круглые скобки. Тело функции пишется с отступом:


def greet():
    print("Привет, мир!")

greet()  # Вызов функции
Важно: Имя функции должно быть осмысленным и описывать её назначение.

6.2. Параметры и возвращаемое значение

Функции могут принимать параметры и возвращать результат с помощью оператора return:


def add(a, b):
    return a + b

result = add(5, 7)

Если функция не содержит return, она возвращает None.

6.5. Рекурсия

Задача 6: Числа Фибоначчи

Реализуйте функцию, которая возвращает N-ное число Фибоначчи:


def fibonacci(n):
    """Возвращает N-ное число Фибоначчи."""
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b

print(fibonacci(10))  # Вывод: 55

В следующей главе мы познакомимся со словарями — мощной структурой данных, которая позволяет хранить пары "ключ-значение".

7. Словари

Словарь — это мощная структура данных в Python, которая хранит пары "ключ-значение". Ключи должны быть уникальными и неизменяемыми (например, строки, числа или кортежи), а значения могут быть любого типа. Словари особенно полезны для работы с данными, где нужно быстро находить значения по ключу.

7.1. Создание словарей

Создать словарь можно несколькими способами:


# Использование фигурных скобок
student = {'name': 'Иван', 'age': 20, 'course': 'Python'}

# Использование функции dict()
credentials = dict(email='user@example.com', password='123456')

# Пустой словарь
empty_dict = {}
Важно: Ключи в словаре должны быть уникальными. Если добавить значение с уже существующим ключом, старое значение будет перезаписано.

7.2. Доступ к элементам

Для доступа к значению по ключу используйте квадратные скобки или метод get():


student = {'name': 'Иван', 'age': 20}
print(student['name'])  # Вывод: Иван
print(student.get('age'))  # Вывод: 20
print(student.get('gender', 'Не указан'))  # Вывод: Не указан

Метод get() позволяет указать значение по умолчанию, если ключ отсутствует.

7.3. Изменение и добавление элементов

Добавление нового ключа-значения или изменение существующего:


student = {'name': 'Иван', 'age': 20}
student['age'] = 21  # Изменение значения
student['city'] = 'Москва'  # Добавление нового ключа
print(student)  # Вывод: {'name': 'Иван', 'age': 21, 'city': 'Москва'}

7.4. Удаление элементов

Удаление элементов из словаря:


student = {'name': 'Иван', 'age': 20, 'city': 'Москва'}
del student['city']  # Удаление по ключу
removed_value = student.pop('age')  # Удаление с возвратом значения
print(removed_value)  # Вывод: 20
print(student)  # Вывод: {'name': 'Иван'}

7.5. Методы словарей

Рассмотрим основные методы работы со словарями:


data = {'a': 1, 'b': 2, 'c': 3}

# Получение всех ключей

# Получение всех значений
values = data.values()  # dict_values([1, 2, 3])

# Получение пар ключ-значение
items = data.items()  # dict_items([('a', 1), ('b', 2), ('c', 3)])

# Очистка словаря
data.clear()
print(data)  # Вывод: {}

def most_frequent_word(text):
    words = text.split()
    frequency = {}
    for word in words:
        frequency[word] = frequency.get(word, 0) + 1
    max_word = max(frequency, key=frequency.get)
    return max_word, frequency[max_word]

text = "python java python c++ java python"
result = most_frequent_word(text)

Задача 3: Инвертирование словаря

def invert_dict(d):
    inverted = {}
    for key, value in d.items():
        inverted[value] = inverted.get(value, []) + [key]
    return inverted

data = {'a': 1, 'b': 2, 'c': 1}
inverted_data = invert_dict(data)
print(inverted_data)  # Вывод: {1: ['a', 'c'], 2: ['b']}
